flow-landing-page

A modern landing page concept for a fictional startup called **Flow**. This project was built to practice structuring real-world marketing websites using semantic HTML and Flexbox-based CSS layouts.

The focus was on clean visual hierarchy, section-based design, and creating a layout that feels like a real SaaS/product landing page.

## ✨ Features

- Clean, modern landing page layout
- Strong visual hierarchy (hero → features → content → footer)
- Flexbox-based layout structure
- Reusable section design approach
- Product-style marketing structure

---

## 🎯 Design Goal

The goal of this project was to simulate a real startup landing page experience. The layout focuses on guiding users through a clear flow of information, similar to modern SaaS websites:

**Hero → Value Proposition → Feature Sections → Call to Action → Footer**
